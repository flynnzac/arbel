3.1 Example: A Line-Oriented Text Editor

' Line-oriented text editor
set /arbed [ registry . ] .
in arbed
   (
     set /buffer [ list . ] . 
     set /cur-line 1 . 
     set /n ( up set /cur-line [ add cur-line 1 . ] . ) .
     set /p ( up set /cur-line [ sub cur-line 1 . ] . ) .
     set /i
         (
           set /arbed-str s buffer .
           in buffer
              (
                to-register [ string-append
                                     "#" [ to-string cur-line . ] . ] .
                set ans arbed-str .
              ) .
         ) .
     set /o
         (
           to-register [ string-append
                                "#" [ to-string cur-line . ] . ] .
           print [ get ans buffer . ] .
         ) .
   ) .

Example of use: 

... go-in arbed .
... i /s "Hello" .
... o .
Hello
... n .
... i /s "Goodbye" .
... o .
Goodbye
... p .
... o .
Hello
