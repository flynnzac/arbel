3.7 Example: Split a String

' Tokenize a string /line by splitting on string /sep
set /tokenize
    (
      set /i 1 .
      set /len [ string-length line . ] .
      set /cur-reg /#1 .
      set /tokened [ list . ] .
      set /init 1 .
      while ( lt-eq i len . )
            (
              set /d [ substring line i i . ] .
              if [ string-eq d sep . ]
                 {
                   in tokened ( set cur-reg buffer . ) .
                   set /cur-reg [ next cur-reg . ] .
                   set /init 1 .
                 }
                 {
                   if [ eq init 1 . ] { set /buffer d . set /init 0 . }
                      { set /buffer [ string-append buffer d . ] . } .
                 } .
              set /i [ add i 1 . ] .
            ) .
      in tokened ( set cur-reg buffer . ) .
      answer tokened .
    )  .

Examples of use: 

... tokenize /line "1,2,3" /sep "," .
ans = a registry with:
#1 of type String, value: 1
#2 of type String, value: 2
#3 of type String, value: 3
