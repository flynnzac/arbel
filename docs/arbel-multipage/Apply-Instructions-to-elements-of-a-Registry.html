4.1.4 Apply Instructions to elements of a Registry

  • ‘do-to-all INSTRUCTION REGISTRY1 REGISTRY2 ... REGISTRYN’ — execute ‘INSTRUCTION’ which takes arguments at Registers ‘/t1’, ‘/t2’, …, ‘/tN’ for each element in ‘REGISTRY1’, …, ‘REGISTRYN’. Set the ‘/ans’ register to a Registry which contains whatever the ‘INSTRUCTION’ evaluates to at the corresponding Registers. For example, ‘do-to-all ( add t1 t2 . ) [ list 1 2 3 . ] [ list 4 5 6 . ]’ sets the ‘/ans’ register to a Registry with elements (5,7,9) at registers ‘(/t1,/t2,/t3)’. ‘do-to-all’ will only return results at Registers that exist in all Registries. So, for example, ‘do-to-all ( add t1 t2 . ) [ list 1 2 3 . ] [ list 4 5 . ]’ sets the ‘/ans’ Register to ‘list 5 7’.
  • ‘collapse INSTRUCTION REGISTRY <STRING>’ — sets the ‘/ans’ register to the value of applying ‘INSTRUCTION’ to the value at Register ‘/STRING1’ and ‘/STRING2’ and then again to the result of that and to the value of ‘/STRING3’ and so on where the ‘INSTRUCTION’ takes each argument at registers ‘/t1’ and ‘/t2’. For example, ‘collapse ( add t1 t2 . ) [ list 1 2 3 . ] "t"’ sets the ‘/ans’ register to ‘6’. If the ‘STRING’ argument is omitted, it is assumed to be ‘"t"’.
