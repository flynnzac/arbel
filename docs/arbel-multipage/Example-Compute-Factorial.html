3.4 Example: Compute Factorial

' compute a factorial using while
set /factorial
    (
      set /i 2 .
      set /prod 1 .
      while ( lt-eq i t . )
            (
              set /prod [ mul i prod . ] .
              set /i [ add i 1 . ] .
            ) .
      answer prod .
    ) .


' compute factorial using recursion

set /factorial-recurse
    (
      if [ eq t 1 . ]
         1
         {
           mul t [ factorial-recurse /t [ sub t 1 . ] . ] .
         } .
    ) .

Example of use: 

... factorial 5 .
ans = 120
