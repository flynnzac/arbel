3.2 Example: Numerical Differentiation

' Compute the numerical derivative of a function
set /deriv
    (
      if [ not [ exist /h . ] . ]
         {
           set /h 0.001 .
         } .
      div [ sub [ f /t [ add x h . ] . ]
                [ f /t [ sub x h . ] . ] . ]
          [ mul 2 h . ] .
    ) .

Example of use: 

... set /f ( power t 2 . ) .
... deriv /f f /x 1.0
ans = 2.0
