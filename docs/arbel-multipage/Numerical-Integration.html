3.5 Numerical Integration

' takes a function as an argument and return left hand
' riemmann sum over range

set /integrate 
    ( 
      set /delta [ div [ sub ub lb . ] n . ] .
      set /x lb .
      set /int 0.0 .
      repeat n
            (
              set /int [ add int [ f /t x . ] . ] .
              set /x [ add x delta . ] .
            ) .
      mul int delta .
    ) .

Example of use: 

... integrate /f ( power t 2 . ) /lb 0.0 /ub 1.0 /n 1000 .
ans = 0.332834
